## Learning Module 4: Coding Agents Everywhere 

### Objective

Scale your AI development workflow from one agent to many: learn to coordinate parallel agents, avoid conflicts, and choose the right tools for spawning asynchronous workflows across code review, communication, and task tracking platforms.

### Lesson Plan
- Asynchronous Coding Agent Workflows
  - How to Become a Multi-Agent Manager
  - Cloud agents
  - Understanding agent isolation (git worktrees and beyond)
  - Agent orchestrator frameworks
- Evaluating the AI Coding Landscape
  - Taxonomy for Understanding Coding Agent Capabilities
- Coding agents for
  - Code review
  - Communications platforms
  - Your task tracking software
