## Learning Module 3: Setting Up an AI-first Development Environment

### Objective

Set up the infrastructure to confidently delegate to one or more agents: verifiable testing environments, well-documented repos, and customized rules that keep agents productive within guardrails.

### Lesson Plan

- Creating Agent-Ready Repositories
    - Setting up verifiable testing environments
    - Design an optimal AGENTS.md
- Customizing Agent Behavior
    - Define custom commands
    - Define custom rules
    - Define subagents
    - Customize your MCP agent ecosystem
